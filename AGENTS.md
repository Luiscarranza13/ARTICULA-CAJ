# Codex Agent Instructions

This project uses reusable skills stored in `.agents/skills`.

When working on tasks:

* Automatically search for relevant skills.
* Read and apply the corresponding `SKILL.md`.
* Combine multiple skills if needed.

Always prioritize:

* clean architecture
* modern UI/UX
* scalable systems
* production-ready code
* accessibility
* responsive design
* security
* performance
* maintainability

Preferred stack:

* Next.js
* React
* TypeScript
* Supabase
* Tailwind
* Framer Motion

Before generating code:

1. Analyze available skills
2. Select the most relevant ones
3. Follow their workflow and conventions
